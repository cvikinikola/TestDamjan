# TestDamjan
 automated UI test using Selenium/Visual Studio 2017 (c#) 
 using System; using System.Threading; using Hanssens.Net; using Microsoft.VisualStudio.TestTools.UnitTesting;using OpenQA.Selenium; using OpenQA.Selenium.Chrome;
 Da biste pokrenuli projekt iz GitHuba, prvo trebate ga preuzeti na računar koristeći Git komandu "git clone" i URL adresu projekta sa GitHubu. Nakon što ste preuzeli projekt, trebali biste otvoriti projekat u  Visual studio 2017 ili 19
Visual studio 2017 ili 19 kliknete u menu dugme Test pa Test Explorer 
U prozorcetu test explorer na UnitTestDamjan desni klik pa Run selected Test i sacekate da odradi testiranje
